Bioskop

Aplikasi layar nirkabel full-stack untuk menampilkan konten HP ke laptop/PC dengan gaya bioskop gelap neon dan nuansa gaming futuristik.

Pilih Mode

Pilih apakah perangkat ini akan menjadi pemancar layar (HP) atau layar penerima (laptop/PC) dalam ekosistem bioskop gelap.

Modus Pemancar

Tekan tombol di bawah untuk mulai mengirimkan layar ponsel. Pastikan browser mendukung getDisplayMedia() dan jalankan lewat HTTPS.

Pastikan browser HP mendukung tangkapan layar dan file ini dibuka melalui HTTPS atau hosting, bukan file://.
Status: Menunggu perintah.
Modus Bioskop

Sambungkan ke pemancar HP. Setelah terhubung, tampilan akan muncul sebagai layar bioskop gelap.

Layar bioskop akan tampil di sini setelah sambungan berhasil.
Status: Menunggu koneksi.

Panduan Pasang

  1. Simpan file ini sebagai index.html.
  2. Buka melalui HTTPS atau hosting, jangan langsung dari file://.
  3. Upload ke GitHub dan aktifkan GitHub Pages lewat Settings > Pages pada branch main atau gh-pages.
  4. Atau deploy ke Netlify dengan menyeret index.html ke dashboard.
  5. Buka URL hasil deploy di HP dan laptop, lalu pilih mode HP pada satu perangkat dan mode Laptop pada perangkat lain.
